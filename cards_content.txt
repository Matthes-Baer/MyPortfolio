| JavaScript:
- ger: JavaScript ist eine vielseitige Programmiersprache, mit der dynamische und interaktive Elemente auf Webseiten erstellt werden können.
- en: JavaScript is a versatile programming language used for creating dynamic and interactive elements on webpages.

| TypeScript:
- ger: TypeScript ist ein stark typisiertes JavaScript-Superset, das den Entwicklungsprozess mittels mehrerer Funktionen verbessert.
- en: TypeScript is a strongly-typed superset of JavaScript that enhances the development process with various features.

| HTML:
- ger: HTML ist eine Auszeichnungssprache, die zur Erstellung und Strukturierung von Webseiten verwendet wird.
- en: HTML is a markup language used for creating and structuring web pages.

| CSS/SCSS:
- ger: CSS ist eine Stylesheet-Sprache zur Beschreibung der visuellen Darstellung. SCSS ist ein Superset von CSS.
- en: CSS is a style sheet language used to describe the visual presentation of HTML or XML documents. SCSS is a superset of CSS.

| React.js:
- ger: React.js ist eine beliebte JavaScript-Bibliothek zur Erstellung von Benutzeroberflächen und wird oftmals mit anderen JavaScript-Bibliotheken verwendet.
- en: React.js is a popular JavaScript library for building user interfaces. It usually gets combined with many other JavaScript libraries.

| Next.js:
- ger: Next.js ist ein beliebtes React-Full-Stack-Framework, das unter anderem serverseitiges Rendering ermöglicht.
- en: Next.js is a popular React full-stack framework that enables server-side rendering, among other features.

| MongdoDB:
- ger: MongoDB ist ein NoSQL-Datenbanksystem, das unter anderem eine hohe Skalierbarkeit für umfangreiche Anwendungen bereitstellt.
- en: MongoDB is a NoSQL database system that provides high scalability, among other features, for large-scale applications.

| Three.js:
- ger: Three.js ist eine JavaScript-Bibliothek zum Rendern von 3D-Grafiken im Browser. In React.js wird sie mit React Three Fiber verwendet.
- en: Three.js is a JavaScript library that is used to render 3D graphics in the browser. In React.js, it's used with React Three Fiber.

| Redux (+ Toolkit):
- ger: Redux ist ein State-Container, der dabei hilft, den State der Anwendung in einem zentralen Speicher zu verwalten.
- en: Redux is a predictable state container for JavaScript applications that helps manage the application's state in a centralized store.
